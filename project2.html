Project 2 – Movie Line (C Program)

Built a C program for COP3502C that simulates customers arriving to multiple movie ticket lines and being processed by multiple booths. The program reads input from a file, assigns customers to lines, and prints the checkout time for each customer.

What the program does

  • Reads number of customers and booths from an input file
  • Assigns customers to one of 12 lines using a rule based on the first letter of their name
  • Uses linked-list queues to model each line
  • Processes customers by earliest arrival time across assigned lines per booth
  • Computes checkout time using: 30 + (tickets × 5)

Skills used

  • C programming
  • Structs and pointers
  • Linked lists & queues
  • File I/O and parsing
  • Debugging and memory management
